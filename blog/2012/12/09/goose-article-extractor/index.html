Goose - Article Extractor

Dec 9th, 2012

In a nutshell, point Goose to an url and it will extract the plain text of the article, along with the main image, embedded movies and meta information that it finds (tags, publish date). Awesome stuff, right?

Checkout the live demo page where you can play with parsing an url of your choice.

Goose is created by Jim Plush from Gravity.com Maven/SBT dependency is available.
